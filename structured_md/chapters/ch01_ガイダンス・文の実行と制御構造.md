  - 授業中に完成した小問については，授業時間中にスタッフ (教員または TA) にチェックを受 けて OK をもらう．スタッフ名をコメントに付記する (出席の確認にも用いる)．
  - 一部途中の場合は，作成途中と付記する．
- 学期終了時: 最終的な成績は最後に提出されたものを (改善コメントを受けて改善した提出があれ ばそちらを) 用いる．

- 提出ファイルは，jupyter から作成した .ipynb ファイルとする．ファイル名を weekN-(学生証番 号).ipynb としておくこと． Colab で演習した場合は，ブラウザ画面左上の File メニューから Download, Download .ipynb と選択して，得られたファイルの名前を整えて提出する．
- 日本語 (または英語) でコメントが付記されていること : { 途中か完成品か，授業中に既に OK を 得ていればその教員名 } 学んだこと，(あれば) 他に参考にした資料や他の人の回答など、(途中で 提出する場合は何がうまくゆかずに困っているなど ) をコメントで盛り込む． (何も苦労がなかっ た場合を除き，自作したことが分かるだけの十分なエピソードを記述すること — 授業中に教員の OK を得た場合は簡潔で良い)
